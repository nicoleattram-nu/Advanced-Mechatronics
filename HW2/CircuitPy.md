Some notes regarding using CircuitPy on the Pico2:

1. Plug in breadboard and open terminal
2. type "screen /dev/tty.u" then tab to complete the port and enter
    - CRTL + C will stop code
    - CRTL + D will start code from the top
    - Press Enter to enter REPL mode (inline python)

To close the screen CRTL +A, CRTL +K, then y
